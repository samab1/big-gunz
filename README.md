# Big Gunz 💪

A dead-simple workout app: two alternating days (A/B), tap any exercise to watch a short
form video right inside the app — no redirect to YouTube.

## How to use

Open `index.html` in any browser (phone works great), or host it with GitHub Pages.

- **Day A — Push & Squat:** goblet squat, incline press, clean & press, fly, halo, dead hang
- **Day B — Pull & Hinge:** kettlebell deadlift, one-arm row, swing, curls, shrugs, dead hang

Train 3 days a week (e.g. Mon/Wed/Fri), alternating A-B-A, then B-A-B the next week.

## The program

Designed for getting bigger with dumbbells, kettlebells (16/20 kg), a bench, and a
pull-up bar. Simple, low-injury-risk movements; progress by adding weight when all sets
feel easy with 2+ reps left in the tank.

No build step, no dependencies — one HTML file.
